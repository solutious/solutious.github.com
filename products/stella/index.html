Stella

FANCY A TEST?

Version 0.6 Follow on GitHub download

 

Run integration tests with beauty and brute strength

Stella is a testing tool. Define testplans using a powerful Ruby DSL* and let Stella take care of the dirty details (like maintaining values between requests).

Sample Test PlanAdd and retrieve a product

testplan :product_api do
  desc "Maintain a value between requests"
  auth :basic, "user", "pass"

  post "/upload" do
    name "Add Product"
    body "bill", "/path/2/file.txt"
    header "X-Stella" => "Version 0.6"
    param :convert => true
    param :rand => rand(100)

    response 200, 201 do |headers, body|
      data = YAML.load(body)
      @product_id = data[:id]    # Save a value
    end
  end

  get "/product" do
    name "View Product"
    param 'id' => @product_id    # Use the saved value

    response 200 do |header, body|
      data = YAML.load(body)
      repeat :times => 2, :wait => 1.second
    end
  end
end
