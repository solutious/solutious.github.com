Stella

FANCY A TEST?

Version 0.7 Follow on GitHub download

 

Perform load tests with beauty and brute strength

Stella is a powerful performance testing tool for web applications. Define test plans using a concise configuration language and Stella takes care of the dirty details (like maintaining values between requests).
